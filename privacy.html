Política de Privacidade

Última atualização: 02 de março de 2026

1. Controlador dos Dados

O responsável pelo tratamento dos dados pessoais é a marca Lúmen · Ichigo Ichie Yume, operada pelo perfil @lumenbrand.

2. Quais Dados Coletamos

2.1 Jukeboxia (este site)

O Jukeboxia é um player musical estático hospedado no GitHub Pages. Não coletamos dados pessoais diretamente nesta plataforma. Dados de acesso padrão (IP, navegador) podem ser registrados pelo GitHub conforme sua própria política de privacidade.

2.2 Lúmen Hub (portal)

No portal Lúmen Hub coletamos dados de identificação (nome, e-mail, OpenID via OAuth), cookies de sessão para autenticação e métricas agregadas de analytics. Consulte a política completa no portal.

3. Cookies

O Jukeboxia não utiliza cookies próprios. O GitHub Pages pode definir cookies técnicos de infraestrutura.
